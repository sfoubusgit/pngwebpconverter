# PNG→WEBP Converter

**⚠️ This is a Python desktop application and cannot be deployed to Netlify Drop.**

## About

A desktop application for batch converting PNG and JPG images to WEBP format. Built with Python, Tkinter, and Pillow.

## Local Installation

### Requirements
- Python 3.x
- Pillow library

### Setup
```bash
pip install -r requirements.txt
```

### Run
```bash
python converter.py
```

Or use the provided batch file:
```bash
start.bat
```

## Files Included

- `converter.py` - Main application code
- `requirements.txt` - Python dependencies
- `start.bat` - Windows launcher script

## Note

This project cannot be deployed to Netlify Drop as it requires a Python runtime environment. It must be run locally on your computer.
